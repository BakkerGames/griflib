# GrifLib

GrifLib is a C# library designed to be used for running games and handling game data.

## Features

- GROD in-memory stackable dictionaries with automatic change tracking
- DAGS scripting language parser and interpreter
- IFGame engine for running interactive fiction text adventures
- IFParser to convert user input into game commands
- IO utilities for handling GRIF input and output file formats

## Names

The name "GrifLib" is derived from "GRIF", which stands for "Game Runner Interactive Fiction". GRIF files are used to store interactive fiction games, and GrifLib provides the necessary tools to run and manage these games.

"GROD" stands for "Game Resource Overlay Dictionary", which is a key component of GrifLib for managing game data.

"DAGS" stands for "Data Access Game Scripts", which is the scripting language used in GrifLib for creating interactive fiction games. DAGS has functions to directly access GROD game data and to perform calculations and manipulations.
